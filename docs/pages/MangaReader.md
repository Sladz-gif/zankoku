# Manga Reader Page (/manga/series/:id)

## Overview
Immersive webtoon-style manga reading experience with vertical scrolling, episode navigation, real-time comments, and progress tracking. Mobile-first design optimized for long-form reading.

## Page Structure

### Header Bar
- **Series Title**: Current manga series name
- **Episode Selector**: Dropdown for episode selection
- **Navigation**: Back button and settings
- **Progress Indicator**: Current reading position

### Main Reading Area
- **Vertical Canvas**: Full-screen reading surface
- **Page Images**: Sequential manga pages
- **Smooth Scrolling**: Continuous vertical navigation
- **Responsive Images**: Optimized for all screen sizes

### Sidebar Features
- **Comments Panel**: Real-time discussion
- **Episode List**: Quick episode navigation
- **Reader Settings**: Display preferences
- **Share Options**: Social sharing tools

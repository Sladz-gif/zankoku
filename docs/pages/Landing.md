## Technical Features

### Animations & Effects
- **Particle System**: 70 animated colored particles floating in background
- **Hover Effects**: Card brightness changes, border color transitions, shadow glows
- **Scroll Animations**: Smooth scrolling to sections
- **Bounce Animation**: Down arrow with continuous bounce effect
- **Gradient Overlays**: Multiple radial gradients for depth

### Responsive Design
- **Mobile Menu**: Hamburger menu for screens < md (768px)
- **Flexible Grids**: 1-column on mobile, 3-4 columns on desktop
- **Scalable Typography**: clamp() functions for responsive font sizes
- **Touch-Friendly**: Large tap targets for mobile devices

### Color Scheme
- **Primary**: Neon purple (#8B00FF)

- **Accent**: Neon red (#FF003C)
- **Gold**: (#FFD700)

# Clans Page (/clans)

## Overview
Comprehensive clan management system featuring clan creation, membership, wars, and team coordination. Strategic team-based gameplay with resource sharing and competitive rankings.

## Page Structure

### Header Section
- **Title**: "CLANS" with Shield icon
- **Subtitle**: "Join forces. Declare war. Dominate together."
- **User Clan Status**: Current clan or "No Clan" indicator

### Tab Navigation
Three main sections for clan interaction:

#### 1. All Clans Tab
**Browse and discover clans**
- **Icon**: Users/Globe
- **Content**: List of all active clans
- **Sorting**: By members, wins, power

#### 2. My Clan Tab
**Manage current clan membership**
- **Icon**: Shield/Home
- **Content**: Current clan details and management
- **Features**: Member list, wars, resources

#### 3. Create Clan Tab
**Start a new clan**
- **Icon**: Plus/PlusCircle
- **Content**: Clan creation form
- **Requirements**: Rank and resource requirements

## Create Clan Section

### Clan Creation Form
**Multi-step clan creation process:**

#### Basic Information
- **Clan Name**: Unique clan identifier
  - Validation: 3-20 characters, unique
  - Restrictions: No profanity, special characters
  
- **Clan Tag**: Short clan abbreviation
  - Format: 2-5 characters
  - Display: Shown next to member names
  
- **Clan Motto**: Clan description/tagline
  - Length: Up to 100 characters
  - Purpose: Attract potential members

#### Clan Settings
- **Recruitment Status**: Open/Closed/Invite-only
- **Minimum Rank**: Required rank to join
- **Faction Preference**: Preferred anime factions
- **Activity Level**: Expected member activity
- **Language**: Primary clan language

#### Clan Customization
- **Emblem Selection**: Choose clan emblem
- **Color Scheme**: Clan color theme
- **Banner Design**: Clan banner customization
- **Description**: Detailed clan information

#### Creation Requirements
- **Player Rank**: Minimum rank 100
- **Resource Cost**: 1000 Silver coins
- **Cooldown**: 7 days between clan creations
- **Verification**: Account verification required

### Clan Management Tools

#### Leader Dashboard
- **Member Management**: Invite, kick, promote members
- **War Management**: Declare and manage wars
- **Resource Control**: Manage clan resources
- **Settings**: Configure clan settings

#### Officer Tools
- **Member Invites**: Invite new members
- **Chat Moderation**: Moderate clan chat
- **Event Planning**: Organize clan events
- **Activity Tracking**: Monitor member activity

### War Rewards

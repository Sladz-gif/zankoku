# Manga Library Page (/manga)

## Overview
Webtoon-style manga discovery platform featuring card-based browsing, advanced search, genre filtering, and Gold-based monetization. Mobile-first design with vertical scrolling optimization.

## Page Structure

### Header Section
- **Title**: "MANGA LIBRARY" with BookOpen icon
- **Subtitle**: "Discover stories from creators worldwide"
- **Search Bar**: Real-time search functionality
- **Navigation**: Tab-based content organization

### Tab Navigation
Four main content categories:

#### 1. Trending Tab
**Focus**: Popular and trending manga
- **Icon**: Fire/TrendingUp
- **Content**: Most popular series this week
- **Sorting**: By views, likes, and recent activity

#### 2. New Tab
**Focus**: Latest releases
- **Icon**: Sparkles/Plus
- **Content**: Recently published series
- **Sorting**: By publication date

#### 3. Popular Tab
**Focus**: All-time favorites
- **Icon**: Star/Heart
- **Content**: Highest rated series
- **Sorting**: By total ratings and views

#### 4. Following Tab
**Focus**: User's followed creators
- **Icon**: User/Bookmark
- **Content**: Updates from followed creators
- **Sorting**: By update chronology

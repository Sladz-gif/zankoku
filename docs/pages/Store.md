# Store Page (/store)

## Overview
In-game store featuring three main categories: Resources, Unlimited Subscriptions, and Content. Real-money transactions with global payment processing and Ghana MoMo support.

## Page Structure

### Header Section
- **Title**: "STORE" with shopping bag icon
- **Subtitle**: "Power your journey with resources and techniques"
- **Navigation**: Tab-based category switching

### Tab Navigation
Three main store categories with distinct icons and themes:

#### 1. Resources Tab
**Theme**: Orange/Purple gradient
- **Icon**: Zap (lightning bolt)
- **Focus**: In-game resource purchases
- **Currency**: Real money → Silver coins

#### 2. Unlimited Tab  
**Theme**: Blue/Cyan gradient
- **Icon**: Infinity symbol
- **Focus**: Subscription-based unlimited access
- **Currency**: Real money → Time-based access

#### 3. Content Tab
**Theme**: Green/Magenta gradient
- **Icon**: BookOpen
- **Focus**: User-generated content access
- **Currency**: Real money → Gold coins
